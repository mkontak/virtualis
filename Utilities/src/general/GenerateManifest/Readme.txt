This utility provides a report to Service about what's installed on the customer's system.
GenerateManifest accepts several parameters that are listed below:

  [/i], [-i], [--input]  - Specifies the input directory.  GenerateManifest takes this folder and generates a report containing the files and their versions inside the specified directory.  "C:\Program Files\Viatronix" is the default.
  [/o], [-o], [--output] - Specifies the output file.      GenerateManifest takes this file and outputs the results to it.  "C:\Temp\[GUID].xml" is the default.
