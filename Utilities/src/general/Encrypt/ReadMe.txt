
Encrypt [-d] [-p PASSWORD] [-f FILE] text

Encrypts/Decrypts the specified text based on the parameters specified. If you want to decrypt you will need a valid
password otherwise you cannot decrypt. If no parameters are specified then the GUI based application will be launched
and you will be able to enter your parameters in there corrseponding text fields.

where:

[-d]                    Indicates that we are to decrypt the text, requires PASSWORD
[-p PASSWORD]           Password required to decrypt
[-f FILE]               Specifies file to write results to
